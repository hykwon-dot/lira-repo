# Simulation Platform Feature Roadmap

## Phase 1 – Intelligent Intake Workspace (현재 작업)
- **사건 진행 타임라인 & 체크리스트**
  - 요약 데이터(IntakeSummary)에서 사건 단계 후보를 추론해 타임라인 카드에 표시합니다.
  - 항목별 완료/보류 상태를 저장하는 클라이언트 상태(Zustand) 추가.
  - `/simulation` 우측 패널에 "타임라인" 카드 배치, 체크/메모 기능 포함.
- **실시간 리스크 알림**
  - Intake 대화 메시지 스트림에서 규칙 기반으로 위험 키워드를 감지.
  - 감지 시 UI 배너와 알림 스택에 등록, 관리자용 `/api/alerts` 엔드포인트로 전달.

## Phase 2 – Evidence Collaboration
- **증거 자료 업로드 & 자동 분류**
  - `/api/evidence/upload` 라우트 생성 및 S3 업로드(사전 서명 URL) 연동.
  - 업로드 후 AI 분석(`/api/evidence/classify`)으로 유형·키워드 태깅 -> 사건 요약과 연동.
  - `/simulation`에 "증거 보관함" 카드 추가, 필터/검색·다운로드 지원.
- **공동 작업 보드 (Collaboration Board)**
  - 조직 멤버/탐정이 메모와 태스크를 남길 수 있는 Kanban 스타일 UI (`/app/simulation/collab`).
  - Prisma에 `CollaborationBoard`, `CollaborationTask` 테이블 추가, 실시간 업데이트(Pusher/Socket) 계획.

## Phase 3 – Performance Intelligence
- **탐정 성과 인사이트 대시보드**
  - `/admin/investigators/insights` 페이지 추가.
  - 성공률, 분야별 매칭 통계, 고객 피드백 요약 시각화(Recharts).
- **맞춤형 온보딩 & 추천**
  - 신규 의뢰인의 업종/목표를 입력 받아 시나리오·탐정·자료를 추천하는 Onboarding Wizard 개편.
  - 추천 결과를 시뮬레이션 초기 상태 및 리포트 탭과 동기화.

## 운영 고려 사항
- AI 호출 증가 대비 Rate-limit & Queue 설계.
- Prisma 마이그레이션: evidence, board, alert, analytics용 테이블 추가.
- 사용자 데이터 보호를 위한 파일 암호화/접근 제어 정책 수립.

> **현재 스프린트 목표:** Phase 1의 타임라인 카드와 리스크 알림 MVP 구현, `/simulation` UI와 자연스럽게 통합.

## AI 확장 아이디어 (제안)

- **대화 요약 → 실행 플랜 자동화**
  - 인테이크 세션 종료 시 AI가 요약과 함께 우선순위 액션 플랜(P0/P1/백업)을 생성해 의뢰인·탐정에게 공유.
  - 후속 리마인더 스케줄링과 관리자 승인 워크플로까지 연동.

- **리스크 트렌드 학습 & 사전 알림**
  - 케이스 유형, 업종, 고객별로 반복되는 위험 패턴을 분석.
  - 유사한 신호가 발생하면 “이전 사건에서 문제됐던 포인트”를 즉시 안내하고 방어 전략을 제시.

- **대화 중 실시간 질문 추천 고도화**
  - 메시지 스트림에서 누락된 필수 정보를 탐지해 즉시 질문 템플릿을 제안.
  - 요약 기반 Next Questions를 강화해 탐정 핸드오프 품질을 높임.

- **멀티모달 증거 정리 보조**
  - 이미지·PDF·음성 등 업로드 자료를 AI가 요약·키워드 추출하여 대화 요약에 자동 첨부.
  - 중요한 증거는 타임라인/체크리스트 항목과 연결해 추적 가능하도록 구성.

- **조사 플로우 시뮬레이터**
  - 케이스 위험도·유형 기반으로 예상 조사 단계, 소요 일정, 리소스를 시뮬레이션.
  - 일정 겹침이나 리소스 부족이 감지되면 관리자에게 재조정 옵션 제안.

- **AI 보조 조사 리포트 초안 작성**
  - 대화 로그, 요약, 증거 자료를 묶어 리포트 초안을 생성하고 탐정이 편집만 하면 되도록 지원.
  - 표준 양식(사건 개요, 수행 내역, 결론, 권고 사항 등)에 맞춰 자동 정리.

- **탐정 매칭 점수 고도화**
  - 탐정별 과거 성과, 전문 분야, 피드백 데이터를 AI 위험 지표와 결합해 정량 매칭 점수 산출.
  - “이번 케이스 성공 확률 92%”처럼 신뢰도 높은 추천 근거를 제공.

- **FAQ 자동 생성 & 셀프서비스 지원**
  - 축적된 대화 로그에서 사건 유형별 FAQ를 도출해 챗봇 셀프서비스에 투입.
  - 반복 문의는 AI가 선 대응하고, 고급 상담만 탐정에게 라우팅.

- **이상 행동 감지 & 내부 알림**
  - 의뢰인/탐정 계정의 비정상 패턴(과도한 취소·고위험 표현 등)을 ML로 감지.
  - 관리자 콘솔과 알림 채널(Slack 등)로 즉시 통보하고 추가 인증 절차를 트리거.

- **성과 분석 & 전략 리포트**
  - 월/분기 단위로 사건 데이터를 분석해 트렌드, 성공률, 리스크 변화를 리포트로 제공.
  - 투자자/경영진 공유용 인포그래픽 및 KPI 대시보드 자동 생성.

## 차세대 AI 혁신 아이디어

- **현장 대응 디지털 트윈 시뮬레이터**
  - 실제 탐정 팀 구성과 장비 보유 현황을 가상 환경에 반영해, 위험 시나리오별 최적 전개 경로를 실시간 재현.
  - 외부 변수(날씨, 위치 기동 시간, 동선 충돌 등)를 자동 반영하여 오프라인 작전의 성공 확률을 예측.
- **협상 스크립트 자동 코치 (구현됨 · 2025-10-11)**
  - 탐정과 의뢰인·피조사자 간 협상 패턴을 분석해 발화별 응답 스크립트와 감정 톤 조절 가이드를 제시.
  - 통화/면담 녹취를 바탕으로 화행 분석, 신뢰도 추정, 리스크 신호 변화를 그래프로 제공.
  - 시뮬레이션 화면에 "협상 스크립트 코치" 패널이 추가되어 실시간 대화 데이터를 기반으로 전략 축, 추천 스크립트, 반론 대응, 톤 가이드를 자동 제공.
- **증거 상관관계 그래프 AI**
  - 업로드된 문서·이미지·로그를 노드 그래프로 시각화하고, 인물/장소/시간 축을 기준으로 자동 클러스터링.
  - 새로운 증거가 추가되면 관련 노드의 강도를 재계산해 "의심 링크"나 "증거 공백"을 즉시 표시.
